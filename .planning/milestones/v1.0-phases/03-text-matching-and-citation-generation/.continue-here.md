The matching pipeline in offscreen.js `matchAndCiteOffscreen`:
1. Build concat string from positionMap entries (with inline wrap-hyphen handling)
2. Try exact `indexOf` match → confidence 1.0 (with multi-occurrence disambiguation)
3. Try `whitespaceStrippedMatch` → strips all whitespace, matches, maps back → confidence 0.99/0.98/0.97 (with multi-occurrence disambiguation)

5. Try `bookendMatch` (if >60 chars) → match first/last 50 chars → confidence 0.92
6. Try fuzzy Levenshtein sliding window (capped at 100 chars) → confidence 0.80+
