# Requirements: Patent Citation Tool

**Defined:** 2026-03-02
**Core Value:** Highlight text on Google Patents, get an accurate citation reference instantly — no PDF downloading, no manual counting.

## v1.1 Requirements

Requirements for v1.1 milestone. Each maps to roadmap phases.

### Silent Mode

- [x] **SLNT-01**: User can enable "silent" trigger mode in popup settings
- [x] **SLNT-02**: When silent mode is active, Ctrl+C on highlighted patent text copies text with citation appended (format: `"text. 4:12-15"`)
- [x] **SLNT-03**: When silent mode is active, patent prefix setting applies to silent citations (format: `"text. '123 Pat., 4:12-15"`)
- [x] **SLNT-04**: When silent mode is active and published application text is highlighted, Ctrl+C appends paragraph citation
- [x] **SLNT-05**: When match confidence is low or no match found, clipboard contains plain text only and a toast explains why

### USPTO Fallback

- [x] **UPTO-01**: Cloudflare Worker is deployed with token validation and CORS handling
- [x] **UPTO-02**: When Google Patents PDF is unavailable, extension fetches patent PDF via Worker proxy to USPTO image server
- [x] **UPTO-03**: USPTO-fetched PDF is parsed identically to Google Patents PDF (same offscreen flow)

### Server Cache

- [x] **CACH-01**: Before fetching PDF, service worker checks Cloudflare KV for cached position map
- [x] **CACH-02**: On cache hit, patent resolves to citation-ready state without PDF download or parse
- [x] **CACH-03**: After successful local parse, position map is uploaded to KV (fire-and-forget)
- [x] **CACH-04**: Cache write checks for existing key first to avoid unnecessary writes (KV quota protection)

## Future Requirements

Deferred to future release. Tracked but not in current roadmap.

### Citation Format

- **CFMT-01**: User can choose citation format (4:5-20 vs col. 4, ll. 5-20 vs column 4, lines 5-20)
- **CFMT-02**: User can define custom citation template string

### Shortcuts

- **SHRT-01**: User can trigger citation via keyboard shortcut (Chrome Commands API)

### Batch Mode

- **BTCH-01**: User can queue multiple citations and copy all at once

### Patent Families

- **PFAM-01**: Continuation patents reuse parent patent's cached position map

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Client-side OCR | Only ~2-5% of patents need it; USPTO fallback solves the problem |
| Non-US patents | Different document formats and citation conventions |
| Citation management | Tool's job ends at the clipboard |
| AI patent summarization | Different product category |
| Always-on copy interception | Must gate on triggerMode === 'silent' and citation availability |
| Raw PDF storage in KV | PDFs are 5-30 MB; store parsed position maps only (10-100 KB) |
| Fallback status indicator | Deferred — not essential for v1.1 |
| Cache status indicator | Deferred — not essential for v1.1 |
