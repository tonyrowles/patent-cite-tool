  <name>Task 1: Create toast UI functions in citation-ui.js</name>
  <files>src/content/citation-ui.js</files>
  <action>
Add two new functions and their CSS helpers to citation-ui.js. Place them after the `showLoadingIndicator()` function and before the CSS functions section.

**1. `showSuccessToast(citation, rect)` function:**

A minimal pill showing just the citation text. Small, unobtrusive, like a confirmation chip.

- Call `getCitationHost()` to get/reset the shadow host
- Create a `<style>` element with success toast CSS (see below)
- Create a `<div>` with class `cite-toast-success` containing the citation text (use `escapeHtml()`)
- Position near bottom of selection: `top = rect.bottom + 6`, `left = rect.left`
- Viewport clamping: if below viewport, show above selection; clamp left to viewport bounds
- Set `host.style.top/left/width/height` (width and height = 'auto')
- Append style and pill to shadow
- `setTimeout(() => dismissCitationUI(), 2000)` for auto-dismiss

**2. `showFailureToast(reason, rect)` function:**

A more prominent pill with the reason text. Larger than success pill, draws attention to the problem.

- Same pattern as success toast but with failure styling
- Uses `cite-toast-failure` class
- Reason text is displayed directly (use `escapeHtml()`)
- `setTimeout(() => dismissCitationUI(), 4000)` for auto-dismiss
- Viewport clamping: wider than success pill (account for ~220px width)
