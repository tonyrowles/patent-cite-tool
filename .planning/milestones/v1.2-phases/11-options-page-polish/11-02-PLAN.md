**Modify `src/popup/popup.js`:**

1. **Remove all settings code** (approximately lines 98-127 — everything after the status `switch` block closing brace that deals with settings). Remove:
   - `const triggerSelect = document.getElementById('triggerMode');`
   - `const displaySelect = document.getElementById('displayMode');`
   - `const patentNumCheckbox = document.getElementById('includePatentNumber');`
   - The entire `if (triggerSelect && displaySelect) { ... }` block including `chrome.storage.sync.get()`, all three `addEventListener('change', ...)` handlers, and the `patentNumCheckbox` conditional

2. **Add the settings link handler** after the status switch block:
   ```javascript
   // Open options page when settings link is clicked
   const settingsLink = document.getElementById('settingsLink');
   if (settingsLink) {
     settingsLink.addEventListener('click', (e) => {
       e.preventDefault();
       chrome.runtime.openOptionsPage();
     });
   }
   ```
   Note: `e.preventDefault()` prevents the `href="#"` from scrolling or causing unwanted navigation.

**What NOT to change:**
- The entire status display logic (DOMContentLoaded, chrome.storage.local.get, patent status switch) remains untouched
- The `<h1>Patent Citation Tool</h1>` heading remains
- The `<div id="content"></div>` remains
- All status CSS classes (.status, .status-ready, .status-fetching, etc.) remain
