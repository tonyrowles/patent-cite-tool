**Create `docs/privacy/` directory** and write `docs/privacy/index.html`.

The privacy policy is a single self-contained HTML page with inline CSS matching the extension's design language. Structure:

1. **HTML head**: charset UTF-8, viewport meta, title "Privacy Policy — Patent Citation Tool", inline `<style>` block.

2. **Inline CSS** — clean, minimal, matching extension style:
   - Body: system font stack (`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`), `font-size: 15px`, `line-height: 1.6`, color `#1e293b`, max-width `680px`, centered with auto margins, padding `48px auto` top/bottom and `0 24px` left/right.
   - h1: `font-size: 24px`, `font-weight: 700`, color `#1e3a8a`, margin-bottom `8px`.
   - h2: `font-size: 18px`, `font-weight: 600`, color `#0f172a`, margin `32px 0 8px`.
   - p, li: color `#334155`, margin-bottom `8px`.
   - strong: color `#0f172a`.
   - a: color `#2563eb`.
   - footer/last-updated: smaller font, `#94a3b8` muted color.

3. **Content sections** — cover all required disclosures:

   **Title**: "Privacy Policy" as `<h1>`, followed by "Patent Citation Tool" in bold and "Last updated: March 2026".

   **Summary**: One paragraph: "Patent Citation Tool does not collect, store, or transmit any personal data. The extension runs entirely in your browser and only communicates with two services for its core citation generation functionality."

   **Section: Information We Collect**: State clearly that NO personal information, browsing history, analytics data, or user account information is collected. The extension does not require sign-in.

   **Section: Services Used**:
   - **Google Patents CDN (patentimages.storage.googleapis.com)**: The extension fetches patent PDF files from Google's public CDN to extract text position data. Only the patent document identifier is included in the request URL. No user data is transmitted.
   - **Cloudflare KV Cache (pct.tonyrowles.com)**: The extension fetches pre-computed patent text position maps from this first-party infrastructure operated by the developer. Only patent document identifiers are sent as part of URL requests. No user data is transmitted. No data is logged or retained beyond the cached position maps.
   - **chrome.storage.sync**: Three user preferences (trigger mode, display mode, patent number prefix toggle) are synced across Chrome sessions using Chrome's built-in sync storage. This data is managed by Chrome's infrastructure and is not accessible to the developer.

   **Section: Data Sharing**: "We do not sell, trade, or transfer any data to third parties. No data is collected that could be shared."

   **Section: Permissions Used**: Brief explanation of why the extension requests its permissions — declarativeContent (activate on patent pages only), offscreen (PDF text extraction), activeTab (read current patent URL), storage (save preferences), contextMenus (right-click citation option), clipboardWrite (copy citations). Each is used solely for citation generation functionality.

   **Section: Changes to This Policy**: Standard statement that the policy may be updated and the date at the top reflects the last revision.

   **Section: Contact**: "For questions about this privacy policy, open an issue at https://github.com/tonyrowles/patent-cite-tool" (link it).

   **Footer**: Small muted text with the extension name and a link back to the Chrome Web Store (or just the GitHub repo).
