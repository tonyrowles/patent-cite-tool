  <name>Task 1: Create esbuild build script and npm scripts</name>
  <files>scripts/build.js, package.json</files>
  <action>
**Create scripts/build.js** using the esbuild JS API. The script should:

1. **Parse CLI args**: Support `--chrome-only`, `--firefox-only`, `--watch` flags. Default (no flags) = both targets, no watch.

2. **Clean step**: `rmSync('dist', { recursive: true, force: true })` before each build.

3. **Chrome build function** (`buildChrome({ sourcemaps = false })`):

   a. **IIFE content bundle**: Single `esbuild.build()` call:
      - `entryPoints: ['src/content/content-script.js']`
      - `bundle: true`
      - `format: 'iife'`
      - `platform: 'browser'`
      - `outfile: 'dist/chrome/content/content.js'`
      - `sourcemap: sourcemaps`
      - Do NOT set `globalName` (pure side-effect IIFE, per anti-pattern guidance)

   b. **ESM bundles**: Single `esbuild.build()` call:
      - `entryPoints: ['src/background/service-worker.js', 'src/offscreen/offscreen.js', 'src/popup/popup.js', 'src/options/options.js']`
      - `bundle: true`
      - `format: 'esm'`
      - `platform: 'browser'`
      - `outbase: 'src'`
      - `outdir: 'dist/chrome'`
      - `sourcemap: sourcemaps`
      - `external: ['../lib/pdf.mjs']` -- CRITICAL: prevents esbuild from bundling the 3MB PDF.js library. The offscreen bundle will import pdf.mjs at runtime from the copied lib/ directory.

   c. Run both esbuild calls with `Promise.all()` for parallel execution.

   d. **Copy static assets** using `fs.cpSync` or `fs.cp` (Node 24 available):
      - `src/icons/` -> `dist/chrome/icons/` (recursive)
      - `src/lib/` -> `dist/chrome/lib/` (recursive -- includes pdf.mjs and pdf.worker.mjs)
      - `src/offscreen/offscreen.html` -> `dist/chrome/offscreen/offscreen.html`
      - `src/popup/popup.html` -> `dist/chrome/popup/popup.html`
      - `src/options/options.html` -> `dist/chrome/options/options.html`

   e. **Transform Chrome manifest**:
      - Read `src/manifest.json`
      - Change `content_scripts[0].js` from the 5-file array to `["content/content.js"]`
      - Write to `dist/chrome/manifest.json` with `JSON.stringify(manifest, null, 2)`
      - Do NOT modify any other manifest fields (especially web_accessible_resources, background, permissions)

4. **Firefox build function** (`buildFirefox()`):
   - `mkdirSync('dist/firefox', { recursive: true })`
   - Copy `src/manifest.firefox.json` to `dist/firefox/manifest.json`
   - That's it -- Phase 16 adds the actual JS bundles

5. **Watch mode function** (`watchChrome()`):
   - Use `esbuild.context()` for both IIFE and ESM configs (same options as build but with `sourcemap: true`)
   - Call `ctx.watch()` on both contexts
   - Copy assets and transform manifest once at start
   - Log "Watching for changes..." to console
   - Register SIGINT handler to call `ctx.dispose()` on both contexts and exit cleanly

6. **Main logic**:
   - Parse args for `--chrome-only`, `--firefox-only`, `--watch`
   - If `--watch`: run watch mode (Chrome only, per user decision)
   - If `--chrome-only`: build Chrome only
   - If `--firefox-only`: build Firefox only
   - Default: build both Chrome and Firefox
   - Log timing: "Built chrome in Xms", "Built firefox in Xms"

**Update package.json scripts**:
Add these scripts (keep all existing scripts):
```json
"build": "node scripts/build.js",
"build:chrome": "node scripts/build.js --chrome-only",
"build:firefox": "node scripts/build.js --firefox-only",
"dev": "node scripts/build.js --watch"
```

**IMPORTANT: Do NOT add esbuild to devDependencies** -- it is already present in node_modules (confirmed installed as 0.27.3). If it's not already in package.json devDependencies, add `"esbuild": "^0.27.0"` to make the dependency explicit.
