Update `scripts/build.js` to produce a complete `dist/firefox/` directory. Read the current file first.

**Add two new config functions:**

`getFirefoxIifeConfig({ sourcemap })` -- Content script bundle for Firefox (same source, different output):
```javascript
{
  entryPoints: ['src/content/content-script.js'],
  bundle: true,
  format: 'iife',
  platform: 'browser',
  outfile: 'dist/firefox/content/content.js',
  sourcemap,
}
```

`getFirefoxEsmConfig({ sourcemap })` -- Firefox ESM bundles. CRITICAL: Use object entry point syntax for background.js so the output path matches the Firefox manifest's `"scripts": ["background/service-worker.js"]`:
```javascript
{
  entryPoints: [
    { in: 'src/firefox/background.js', out: 'background/service-worker' },
    { in: 'src/popup/popup.js', out: 'popup/popup' },
    { in: 'src/options/options.js', out: 'options/options' },
  ],
  bundle: true,
  format: 'esm',
  platform: 'browser',
  outdir: 'dist/firefox',
  sourcemap,
  external: ['../lib/pdf.mjs'],
}
```

NOTE: Do NOT use `outbase` here -- the object entry point syntax explicitly controls output paths. With outbase:'src', `src/firefox/background.js` would output to `dist/firefox/firefox/background.js` which is WRONG (see RESEARCH.md Pitfall 3).

**Add `copyFirefoxStaticAssets()` function:**
Mirror `copyStaticAssets()` for Firefox target:
- Copy `src/icons` -> `dist/firefox/icons` (recursive)
- Copy `src/lib` -> `dist/firefox/lib` (recursive) -- CRITICAL: pdf.mjs + pdf.worker.mjs must be in dist/firefox/lib/ for PDF.js to work
- Copy `src/popup/popup.html` -> `dist/firefox/popup/popup.html`
- Copy `src/options/options.html` -> `dist/firefox/options/options.html`
- Ensure output directories exist with `mkdirSync({ recursive: true })`
- Do NOT copy offscreen.html -- Firefox has no offscreen document

**Update `buildFirefox()` to async:**
```javascript
async function buildFirefox({ sourcemaps = false } = {}) {
  const start = Date.now();

  fs.mkdirSync('dist/firefox', { recursive: true });
  fs.copyFileSync('src/manifest.firefox.json', 'dist/firefox/manifest.json');

  await Promise.all([
    esbuild.build(getFirefoxIifeConfig({ sourcemap: sourcemaps })),
    esbuild.build(getFirefoxEsmConfig({ sourcemap: sourcemaps })),
  ]);

  copyFirefoxStaticAssets();

  const elapsed = Date.now() - start;
  console.log(`Built firefox in ${elapsed}ms`);
}
```

**Update `main()` to await buildFirefox:**
Change `buildFirefox();` to `await buildFirefox();` in the default (build both) branch and the `firefoxOnly` branch.
