baseline.json shape (per-entry):
```json
{ "<caseId>": { "citation": "1:26-27", "confidence": 0.98 } }
```

NOTE: baseline.json stores NUMERIC confidence (0.0-1.0). The regression spec converts numeric → color via:
- >= 0.95 → green
- >= 0.80 → yellow
- else → red

The capture script must therefore convert observed COLOR back to a representative numeric value:
- green → 0.98 (the midpoint of [0.95, 1.0] that the existing baseline uses)
- yellow → 0.90 (the midpoint of [0.80, 0.95])
- red → 0.70 (representative)

This preserves the asserted-color semantics. Any case that currently passes assertion with a specific numeric was authored by a human; the recalibration must NOT touch the 43 already-passing entries.
