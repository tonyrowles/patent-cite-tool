atomicWriteJson pattern to inline verbatim (tests/e2e/lib/llm-report.js lines 76-91):
```
function atomicWriteJson(destPath, content) {
  const tmpPath = `${destPath}.tmp.${process.pid}`;
  fs.writeFileSync(tmpPath, content);
  try {
    fs.renameSync(tmpPath, destPath);
  } catch (err) {
    if (err && err.code === 'EXDEV') {
      fs.writeFileSync(destPath, content);
      try { fs.unlinkSync(tmpPath); } catch { /* best-effort */ }
      return;
    }
    throw err;
  }
}
```

Source llm-report.json iteration shape (tests/e2e/lib/llm-report.js + the re-stamped fixture):
- iteration_n: number
- iso: ISO string
- classification: one of WRONG_CITATION | VERIFIER_DISAGREE | HARNESS_ERROR | LLM_API_ERROR | LLM_HALLUCINATED_SELECTION | PASS
- llm_selection: { caseId, patentId, selectedText, ... } | null (null on pre-browser failure paths)
- citation: "C:L1-L2" string | null
- verifier_verdict: { status: 'pass'|'fail', tier_used: 'A'|'B'|'C'|null, reason: string, ... } | null
- scroll_y, viewport_width, viewport_height, selected_node_xpath: present per D-13, values may be null

D-01 eligibility: classification ∈ {WRONG_CITATION, VERIFIER_DISAGREE} ONLY.
D-02 ineligibility: HARNESS_ERROR, LLM_API_ERROR, LLM_HALLUCINATED_SELECTION, PASS → NOT_REPLAYABLE with reason = classification.
D-03 confirm = string equality on verifier_verdict.status between original and each replay.
D-04 Tier C unanimous COUNTS as CONFIRMED (no tier_used branching in this module).
D-08 no determinism guards (no seed, no env scrub).

D-09 top-level shape:
```
{
  schema_version: 1,
  source_llm_report: <string path>,
  run_id: <string copied from source>,
  started_iso: <ISO>,
  finished_iso: <ISO>,
  summary: { confirmed_count: N, flake_count: N, not_replayable_count: N },
  replays: [...]
}
```

D-10 per-replay entry shape:
```
{
  iteration_n: <int>,
  original_verdict_status: <string|null>,
  runs: [{ status, tier_used, reason }, ...],   // length 0 when NOT_REPLAYABLE
  confirmed_count: <int>,
  total_runs: <int>,
  verdict: 'CONFIRMED' | 'FLAKE' | 'NOT_REPLAYABLE',
  reason: <string>  // present ONLY when verdict === 'NOT_REPLAYABLE'
}
```
