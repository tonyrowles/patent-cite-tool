5-state truth table (LOCKED per 45-CONTEXT D-04 + FLAKE-01):
  FLAKE_SUPPRESSED  = suppressions[fingerprint] exists AND new Date(suppressions[fingerprint].until) > now()
                       → action: 'skip', returned BEFORE any outcomes analysis
  CONFIRMED_BUG     = last 10 outcomes all 'fail' (zero pass) AND last 3 all 'fail'
                       → action: 'auto-fix'
  LIKELY_BUG        = failures >= 7 in last 10 outcomes (and NOT CONFIRMED_BUG)
                       → action: 'auto-fix'
  INTERMITTENT      = failures in {4,5,6} in last 10 outcomes
                       → action: 're-quarantine'
  FLAKE_ESCALATION  = failures <= 3 in last 10 outcomes AND (recentFlakes within 14d + 1) >= 3
                       → action: 'open-flake-investigation', also returns `until = now + 30d` for the caller to write
  FLAKE             = failures <= 3 in last 10 outcomes AND (recentFlakes within 14d + 1) < 3
                       → action: 're-quarantine'
