Bypass detection algorithm (RESEARCH.md lines 631-649):
```
For each verifier-gate run R with R.head_branch matching /^auto-fix\//:
  PR_NUM = R.pull_requests[0].number
  PR_MERGED_AT = pr.merged_at  (via `gh api .../pulls/$PR_NUM --jq .merged_at`)
  R_COMPLETED_AT = R.updated_at  (proxy for completion)
  BYPASS_DETECTED = (R.conclusion in {'success','failure'} AND R_COMPLETED_AT > PR_MERGED_AT)
  Edge: cancelled/skipped → BYPASS_DETECTED=true (gate never ran)
  Edge: pr not merged → skip
  Edge: run not completed → BYPASS_DETECTED=null (defer)
```
