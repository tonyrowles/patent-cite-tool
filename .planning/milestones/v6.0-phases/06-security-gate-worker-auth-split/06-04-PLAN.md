- Generate: NEW_TOKEN=$(openssl rand -hex 32)
- Cloudflare: cd worker && echo "$NEW_TOKEN" | wrangler secret put PROXY_TOKEN   (requires the user's Cloudflare auth)
- GitHub: create/update the PROXY_TOKEN repository secret with the same value (repo admin, browser)
- Local build token: write NEW_TOKEN into a git-ignored root .env (PROXY_TOKEN=...) — .env is gitignored by 06-01
- Rebuild: PROXY_TOKEN=$NEW_TOKEN npm run build
- Gate: grep -rn '4509b9943f' src/  → zero;  grep -rn "$NEW_TOKEN" src/  → zero (value only in dist/, .env, and the live secrets)
- The Worker reads env.PROXY_TOKEN from the Wrangler secret binding; no worker code change is needed for rotation.
