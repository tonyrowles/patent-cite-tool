CROSS-COLUMN LONG-FORM (RESOLVED — research A1 left it open; this plan locks it):
formatCitationLong cases —
  - same column AND same line: `Col. {col}, l. {line}`
  - same column, line range:    `Col. {col}, ll. {startLine}-{endLine}`
