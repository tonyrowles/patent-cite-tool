<objective>
Close (or formally defer) gap_inventory Bucket B (7 TIMEOUT_PILL cases).

For these 7 cases, selectText successfully selects text in the live Google Patents HTML, but the extension's offscreen pipeline returns "Text not found in patent specification" — the citation pill never attaches and the spec times out at 30s. The 7 cases (per 27-05-SUMMARY.md):
- US11427642-claims-1
- US11427642-repetitive
- US4723129-claims
- US5371234-chemical-cross-col
- US5371234-claims
- US7346586-claims-repetitive
- US8352400-claims (the long-known smoke failure)

**Two failure sub-classes (must triage per case):**
1. **Data sub-class — HTML↔PDF text bridge fixable:** The HTML form differs from the PDF form in a way that the regen script's normalizer could not bridge (e.g., a claim preamble present in PDF but stripped in HTML, or a chemical SMILES string formatted differently). Fix: refine the needle so the SAME selection text exists in both HTML and PDF fixtures.
2. **Extension sub-class — beyond test data:** The PDF parsing pipeline (offscreen.js + matching.js) cannot locate the text regardless of needle form (e.g., OCR artifacts in the PDF, gaps in the PositionMap). Fix: NOT in this plan's scope. Skip with a documented deferral to Phase 28 (which will independently re-parse the PDF and adjudicate).

**Why this plan needs depends_on: [27-07, 27-08]:** Both 27-07 and 27-08 also touch `tests/test-cases.js` or `tests/e2e/specs/regression.spec.js`. To avoid same-wave file conflicts and merge-time chaos, 27-09 runs in Wave 2 (after 27-07 and 27-08 land).

**Why autonomous: false:** This plan contains a CHECKPOINT after the per-case triage. The orchestrator must show the user the triage verdict (which cases got data-fixed vs deferred) before the deferred cases are skipped in the production spec. The user may have product-level preferences about whether to skip vs investigate immediately.

Purpose: SEL-03 contract says "every patent matches baseline." TIMEOUT_PILL cases that cannot be data-fixed cannot satisfy SEL-03 within Phase 27's test-only scope (no production code changes). Formal deferral with audit trail keeps the contract honest.

Output: Per-case verdict in 27-09-SUMMARY.md. tests/test-cases.js updated for data-fix cases. tests/e2e/specs/regression.spec.js gets a TIMEOUT_PILL_DEFERRED skip path for extension-sub-class cases, pointing at Phase 28.
</objective>
