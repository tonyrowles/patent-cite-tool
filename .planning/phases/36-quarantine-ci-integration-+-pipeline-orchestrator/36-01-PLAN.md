<objective>
Ship the non-gating quarantine Playwright spec and its npm script. The spec mirrors `regression.spec.js` but iterates the (currently empty) `TEST_CASES_QUARANTINE` corpus, one `test()` per entry, and emits a per-case RPT-01 `report.json` so the nightly issue filer (Plan 36-03 / 36-04) has per-case detail. The `e2e:quarantine` npm script runs it with `--retries=0` (overriding the config's `retries: CI ? 1 : 0`, D-02) and `--pass-with-no-tests` (so the empty corpus exits 0 with 0 tests, D-03/SC-1).

Purpose: QUAR-03 — give the nightly a runnable quarantine project that is byte-safe against the empty corpus today and ready to surface real failures as the corpus grows.
Output: `tests/e2e/specs/quarantine.spec.js` (new), `package.json` `e2e:quarantine` script (new).
</objective>
